my-fuzz
=========



- `source <https://github.com/fanhouin/my-fuzz/tree/main>`_

- depend on

  - `getkin / kin-openapi <https://github.com/getkin/kin-openapi>`_

    - A Go project for handling OpenAPI files. Target the latest OpenAPI version (currently 3), but the project contains support for older OpenAPI versions too.


Go Package
-------------

- `openapi3 <https://pkg.go.dev/github.com/getkin/kin-openapi/openapi3?utm_source=godoc>`_

  - Package openapi3 parses and writes OpenAPI 3 specification documents.
